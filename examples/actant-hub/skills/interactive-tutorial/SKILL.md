# Interactive Tutorial

## Overview

Practice-driven tutorials to help users master Actant core features.

## Progressive Skill Building

- **Basic**: Installation, first command, simple task delegation
- **Intermediate**: Skills, workflows, multi-Agent coordination
- **Advanced**: Custom skills, Hub management, pipeline orchestration

## Hands-On Exercises

- Each lesson: explain concept → demonstrate → user tries → verify
- Provide starter files or commands; user completes with guidance
- Offer hints on struggle; avoid giving full answer too early

## Real-Time Feedback

- Validate user actions (e.g., correct command, expected output)
- Explain errors in tutorial context; suggest fix
- Celebrate completion; unlock next lesson

## Adaptive Difficulty

- Adjust pace based on user performance (fast/slow)
- Skip lessons user already demonstrates mastery
- Offer challenge variants for advanced learners

## Completion Tracking

- Persist progress (lessons completed, scores)
- Show progress bar or checklist; suggest next lesson
- Award badges or certificates for milestones
