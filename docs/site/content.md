# Actant Site Content

当前站点文案只应表达一件事：

- Actant 正在收敛为面向 Agent 的 `ContextFS`
- 当前优先级是统一上下文表示、寻址、操作、权限与生命周期
- `VFS Kernel` 是实现内核，不是产品名称

## Public Message

Actant is resetting its architecture around ContextFS.

The current public baseline is:

- Product layer: `ContextFS`
- Implementation layer: `VFS Kernel`
- Core objects: `Project`, `Source`, `Capability`
- V1 built-ins: `SkillSource`, `McpConfigSource`, `McpRuntimeSource`, `AgentRuntime`
- V1 operations: `read`, `write`, `list`, `stat`, `watch`, `stream`

## Public Warning

The repository is in a documentation-first reset stage.
Historical runtime narratives, workflow-centric models, and DomainContext-era descriptions are not current.
