# Human Documentation

This directory is exclusively for human-authored content. AI agents must never auto-generate or auto-modify files here.

## Subdirectories

- `reviews/` — Human code review notes and feedback
- `meetings/` — Meeting notes, discussion outcomes, and decisions

## Top-Level Files

- `onboarding.md` — New developer onboarding guide
- `roadmap.md` — Human-maintained project roadmap and priorities

## File Naming

Use `YYYY-MM-DD-topic.md` format for dated entries.
